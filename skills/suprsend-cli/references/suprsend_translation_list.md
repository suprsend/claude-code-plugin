# suprsend translation list

List Translations

```
suprsend translation list [flags]
```

### Options

```
  -c, --content string   Include content in the output (default "false")
  -h, --help             help for list
  -l, --limit int        Limit the number of translations to list (default 20)
  -m, --mode string      Mode to list translations for (default "live")
  -f, --offset int       Offset the number of translations to list
  -o, --output string    Output type (pretty, yaml, json) (default "pretty")
```

### Options inherited from parent commands

```
      --config string          config file (default: $HOME/.suprsend.yaml)
  -n, --no-color               Disable color output (default: $NO_COLOR)
  -s, --service-token string   Service token (default: $SUPRSEND_SERVICE_TOKEN)
  -v, --verbosity string       Log level (debug, info, warn, error, fatal, panic) (default "info")
  -w, --workspace string       Workspace to list translations for (default "staging")
```
