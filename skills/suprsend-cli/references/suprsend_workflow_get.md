# suprsend workflow get

Get workflow details

Get workfow details of a specific wf. Example: suprsend workflow get <slug>

```
suprsend workflow get [flags]
```

### Options

```
  -h, --help          help for get
      --mode string   mode to fetch worklfow from. (default "live")
```
